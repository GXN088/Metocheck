# Metocheck
Создайте программу, которая печатает персонализированное расписание мероприятий. Создайте и используйте эти три метода:
1.Создайте метод с именем printHeader(), который печатает:
=================
Event Schedule
=================
2.Создайте метод с именем printEvent(), который печатает:
>> Main Event <<
Time: 7:00 PM
Location: Hall A
3.Создайте метод с именем printFooter(), который печатает:
=================
Thank You!
=================
Программа должна напечатать полное расписание, вызывая эти методы в правильном порядке:Сначала заголовокДетали события три разаФутер последним
